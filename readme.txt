■ゲームの開始方法
1）RPGツクール2000 RTP　が必要です。以下の場所からダウンロードし、インストールしてください。
http://www.famitsu.com/freegame/rtp/
2）フォルダの中にある　start.exe　か　start_window.exe　を実行してください。


■バージョンアップする場合
・通常版
新しいバージョンのファイルを解凍し、
gameフォルダ内にセーブファイル（Save**.lsd）をコピーしてから、
ゲーム開始後の通常移動画面でShiftキーを押してサブメニューを開いて下さい。
バージョンアップ後に不具合が出た場合も、
Shiftキーを押すことで修正される場合があります。

・パッチ版
フォルダ内の全ファイルを旧verに上書きして下さい。
それ以外は通常版と同じです。


■操作方法
方向キー［↑］［↓］［←］［→］
　：カーソルの移動、選択肢の操作
決定キー［Enter］［Space］［Ｚ］
　：ポイントの選択、選択肢の決定、押しっぱなしでカーソル加速
取り消しキー［Esc］［Ｘ］［Ｃ］［Ｖ］［Ｂ］［Ｎ］［0］
　：メニューの呼び出し、キャンセル
［Shift］
　：サブメニューを開く
　　イベント中に押すと文字表示速度を切り替える
［F4］
　：全画面表示（フルスクリーンモード）／ウィンドウ表示（ウィンドウモード）との切り替え
［F5］
　：ウィンドウモード時、ウィンドウサイズ（ダブルサイズ／シングルサイズ）の切り替え
［F12］
　：タイトル画面に戻る

文章表示中にShiftをしばらく押しっぱなしにした状態で決定キーを押す
　：文章スキップ

■遊び方
・基本
　カーソル(白い四角)を、ダンジョン内の丸いポイントに合わせて調べるとストーリーが進み、
調べられるポイントが増えます。
　マップを埋めながらダンジョンを探索しましょう。
